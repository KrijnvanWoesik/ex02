# Blinky, Hello world in VHDL

Blink a led with a 1 second interval.

## Function description:

Hello world in VHDL. 

## Input- output diagram
```
       +----------+
       |          |
       |          |   
clk -->|  Blinky  |-----> LED blinking at 1 second interval
       |          |
       |          |
       +----------+
```
